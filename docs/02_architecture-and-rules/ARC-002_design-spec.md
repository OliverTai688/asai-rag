# **一、設計理念  Design Philosophy**

| **如深海浮月，以誠意為底色，以智慧為光澤***Like a moon rising from the deep sea — sincerity as foundation, wisdom as shimmer.* |
| --------------------------------------------------------------------------------------------------------------------------------------- |

*誠問 AI 的視覺語言汲取自一幅畫面：深藍海底中，大小不一的玻璃圓球靜靜漂浮，每一顆都折射著金色光暈。藍色代表沉穩與信任，白色帶來清晰與空間感，而金色則是智慧與價值的點睛之筆。*

## **1.1 三大設計原則**

| **清晰  Clarity** 以大量留白與清晰的資訊層次，讓業務員在任何情境下都能快速找到所需。白色主體傳遞開放與透明。 | **信任  Trust** 深藍色是金融與專業服務的通用語言，代表誠問AI 的專業、可靠與深度。藍色主導視覺。 | **價值  Value** 金色作為畫龍點睛的高光，用於關鍵按鈕、邊框、徽章，點出誠問AI 帶給業務員的智慧增值。 |
| ------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |

## **1.2 視覺風格對照**

| **✗  避免的視覺方向** | **✓  目標視覺方向**       |
| ----------------------------- | --------------------------------- |
| 純白+ 彩色扁平設計，缺乏質感  | 白色主體+ 深藍標題，金色輕點綴    |
| 過度使用紫色漸層（AI 氾濫感） | 藍白雙色為主，色彩克制有力        |
| 深色底黑暗主題，閱讀壓力大    | 淺色主體，深藍作強調區，舒適易讀  |
| 卡通圖示、過度活潑動畫        | 幾何線條圖示，精準且有份量        |
| 字體過細、缺乏權威感          | 中粗字重，數位閱讀最佳易讀性      |
| 顏色過多、視覺雜亂            | 三色系統（藍/ 白 / 金），清晰聚焦 |

# **六、動態規範  Motion & Animation**

## **6.1 動畫設計原則**

動畫存在的目的是「引導注意力」與「傳達狀態」，不是裝飾。所有動畫必須能說出存在的理由。

**提示  **凡是 prefers-reduced-motion 媒體查詢成立時，所有移動類動畫縮減為純 opacity 淡入淡出。

## **6.2 緩動函數 Easing Tokens**

| **--ease-out**    | cubic-bezier(0, 0, 0.2, 1) —— 元素進入場景，自然減速               |
| ----------------------- | -------------------------------------------------------------------- |
| **--ease-in**     | cubic-bezier(0.4, 0, 1, 1) —— 元素離開場景                         |
| **--ease-inout**  | cubic-bezier(0.4, 0, 0.2, 1) —— 狀態切換、hover 過渡               |
| **--ease-spring** | cubic-bezier(0.175,0.885,0.32,1.275) —— 按鈕彈性、Notification pop |

## **無障礙設計要求**

● 所有可互動元素必須有可見的 Focus Ring（3px 藍色或金色外框）

● 圖標必須附 aria-label 或配合文字說明，不可純圖標

● 色彩不得作為傳遞資訊的唯一手段（需配合形狀、文字）

● Prefers-reduced-motion 媒體查詢：所有移動動畫縮減為 opacity-only

● 字體最小值：正文 14px、輔助文字 12px，禁止使用 11px 以下

● 行距（line-height）：正文至少 1.5，標題至少 1.3
