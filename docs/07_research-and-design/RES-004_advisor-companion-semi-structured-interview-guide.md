**顧問陪談・半結構訪綱**

顧問 × 保險業務員　|　協助業務員看懂客戶輪廓、想清楚怎麼跟客戶對話

業務員不需要懂 SPIN，只要回答顧問的問題即可

# **這份訪綱在做什麼**

這是一份「顧問訪談業務員」的半結構訪綱。在業務員真正去拜訪客戶之前，顧問先用這份訪綱跟業務員對談，透過一連串白話問題，帶業務員把客戶的輪廓拼出來，並想清楚開場、切入點與該注意的地方。

| 關鍵設計：SPIN 藏在顧問心裡　顧問是用 SPIN 四階段（情境→問題→影響→需求）來安排提問順序的，但對業務員只用白話發問。所以業務員就算完全沒聽過 SPIN，照樣能在過程中被一步步引導，最後得到「客戶輪廓」與「對話準備」兩項成果。對應關係見第二頁的對照表。 |

# **半結構標示與隱藏的 SPIN 對照**

本訪綱為半結構：核心題每次必問，追問依業務員的回答彈性使用，順序可調。下表同時呈現顧問心中的 SPIN 階段如何對應到實際問業務員的白話問題。

| 顧問心中（SPIN） | 實際問業務員的白話問題 | 這一步幫業務員得到 |
| :---- | :---- | :---- |
| **Situation 情境** | 「客戶現在的狀況你了解多少？」 | 客戶輪廓的事實面、找出還不確定的地方。 |
| **Problem 問題** | 「以他的狀況，你覺得他可能在煩惱什麼？」 | 客戶的痛點與可切入的關心點。 |
| **Implication 影響** | 「這些如果不處理，會怎麼樣？」 | 對話時可用來引導客戶思考的素材。 |
| **Need-Payoff 需求** | 「如果幫他解決，他最在意什麼？」 | 對話收尾方向與想取得的下一步。 |
| **（收斂）** | 「那你打算怎麼跟他聊？」 | 把以上整理成具體的對話準備卡。 |

| 四個進行原則　① 全程用白話，不對業務員提「SPIN」這個詞。② 多用「你覺得呢？」讓業務員自己想，顧問再補。③ 業務員答不出來的，就是他下次拜訪要去問客戶的。④ 最後一定要把對話準備卡填出來，才算完成。 |
| :---- |

  **第 0 段｜先確定我們在談誰**  

*（暖身・聚焦）*

**這一段要幫業務員：**鎖定一位具體客戶，了解業務員的起點與最沒把握的地方。

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **你這次想準備拜訪的是哪一位客戶？你跟他的關係是？** |  |
| **◆** **核心題** **（必問）** | **關於「要怎麼跟他開口、聊什麼」，你現在最沒把握的是哪一塊？** |  |
| **↳** **追問** **（選用）** | 為什麼想找這位客戶談？最近有什麼契機嗎？ |  |
| **↳** **追問** **（選用）** | 你之前跟他接觸過嗎？上次大概聊到哪裡？ |  |

| 顧問引導重點　先讓業務員放鬆、講出真正的卡點。他越誠實說出「我不知道怎麼開口」，後面越好幫。 |
| :---- |

  **第 1 段｜先把客戶現況拼出來**  

*（顧問框架：Situation 情境）*

**這一段要幫業務員：**協助業務員整理客戶的家庭、工作、財務與現有保障，分出「已知」與「待確認」，這就是客戶輪廓的事實面。

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **他的家庭大概是什麼狀況？誰是家裡的經濟支柱？ （要載入家庭結構圖，並且要包含前夫前妻，載入台灣關係的描述）** |  |
| **◆** **核心題** **（必問）** | **他的工作跟收入，你知道多少？** |  |
|  | **對於職涯的階段或規劃？如三年後要退休的情境** |  |
| **◆** **核心題** **（必問）** | **他現在有哪些保險？內容你清楚嗎？** |  |
| **↳** **追問** **（選用）** | 這些裡面，哪些是你確定的，哪些其實是你猜的？ |  |
| **↳** **追問** **（選用）** | 那些「還不確定」的，下次見面可以怎麼自然問出來？ |  |

| 顧問引導重點　邊聽邊幫業務員把「已知 vs 待確認」分兩欄。待確認的，就是他下次拜訪要蒐集的清單。 |
| :---- |

  **第 2 段｜他可能在煩惱什麼**  

*（顧問框架：Problem 問題）*

**這一段要幫業務員：**協助業務員從客戶現況推測潛在的困擾與風險，找出可以自然關心、切入的點。

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **以你對他的了解，他生活裡可能有哪些煩惱或壓力？** |  |
| **◆** **核心題** **（必問）** | **有哪些風險，是他自己可能還沒意識到的？** |  |
| **↳** **追問** **（選用）** | 他最近生活有沒有發生什麼事，可能讓他開始在意這些？ |  |
| **↳** **追問** **（選用）** | 這些煩惱裡，你覺得哪一個他最有感？ |  |
| **↳** **追問** **（選用）** | 你會怎麼讓他「自己講出」這個煩惱，而不是你直接點破？ |  |

| 顧問引導重點　提醒業務員：讓客戶自己說出煩惱，遠比業務員幫他指出更有效，也比較不會引起防備。 |
| :---- |

  **第 3 段｜如果不處理會怎樣**  

*（顧問框架：Implication 影響）*

**這一段要幫業務員：**協助業務員想清楚客戶的問題若不處理的後果，作為對話時引導客戶思考的素材（不是用來嚇客戶）。

\- 有沒有需要PQ表? 量化、質化

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **如果這些煩惱一直沒處理，對他和家人可能造成什麼影響？** |  |
| **◆** **核心題** **（必問）** | **這些影響裡，哪一個是他最不希望發生的？** |  |
| **↳** **追問** **（選用）** | 這些影響牽涉到他最在意的誰？（太太、孩子、父母） |  |
| **↳** **追問** **（選用）** | 對話時，你可以怎麼「用問的」讓他自己想到這些，而不是你直接說？ |  |

| 顧問引導重點　強調：影響是用來幫客戶看清楚、自己有感，絕不是製造恐懼或壓迫。 |
| :---- |

  **第 4 段｜幫他解決後，他會在意什麼**  

*（顧問框架：Need-Payoff 需求）*

**這一段要幫業務員：**協助業務員想清楚客戶真正想要的安心與價值，作為對話收尾的方向。

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **如果這些問題都被妥善解決，他最想得到的是什麼感覺或結果？** |  |
| **◆** **核心題** **（必問）** | **什麼樣的規劃，會讓他覺得「這真的有幫到我」？** |  |
|  | **那目前除了保單規劃本身，身邊有沒有後勤資源可以服務客戶？** | 這題可以作為誠交俱樂部後勤資源的介紹 |
| **↳** **追問** **（選用）** | 他做決定時，會不會需要跟誰商量？（例如配偶） |  |
| **↳** **追問** **（選用）** | 對話的最後，你希望他答應你的「下一步」是什麼？ |  |

| 顧問引導重點　提醒：價值要讓客戶自己說出口；同時陪業務員先想好他要爭取的「下一步承諾」。 |
| :---- |

  **第 5 段｜那你打算怎麼跟他聊**  

*（收斂・對話準備）*

**這一段要幫業務員：**把前面釐清的一切，收斂成具體的開場、切入點與注意事項——也就是對話準備卡。

| 類型 | 顧問問業務員的問題（白話） | 業務員回答記錄 |
| :---- | :---- | :---- |
| **◆** **核心題** **（必問）** | **根據我們剛剛談的，你打算怎麼開場？（先想一個「不談商品」的版本）** |  |
| **◆** **核心題** **（必問）** | **哪些話題可以幫你自然破冰、拉近距離？** |  |
| **◆** **核心題** **（必問）** | **跟這位客戶相處，有什麼要特別注意或避免的？** |  |
| **↳** **追問** **（選用）** | 你準備好「先問哪三個問題」了嗎？ |  |
| **↳** **追問** **（選用）** | 如果他防備、不太想聊，你會怎麼回應？ |  |

| 顧問引導重點　陪業務員把開場白實際講一遍、順一遍，並把答案直接謄到下一節的對話準備卡。 |

# **產出一：客戶輪廓表（範例已填寫）**

談完前五段後，顧問與業務員一起把答案整理成下表。左欄固定，右欄為範例客戶「陳志明」的填寫示意，實際使用時依真實客戶填入。

| 輪廓項目 | 內容（範例：陳志明） |
| :---- | :---- |
| **客戶與關係** | 陳志明，38 歲，朋友介紹的潛在客戶。 |
| **家庭狀況** | 已婚；太太兼職；女兒 7 歲、兒子 4 歲；父母住附近，母親健康轉差。 |
| **工作與收入** | 科技業產品經理，是家庭主要經濟支柱，家庭年收入約 180 萬。 |
| **現有保障** | 年輕時的儲蓄險＋意外險各一張、公司團保（內容業務員不確定）。 |
| **已確認** | 家庭結構、他是主要收入者、有房貸。 |
| **待確認（下次要問）** | 保單實際內容、房貸金額、太太與孩子的保障、退休與教育金規劃。 |
| **可能的煩惱／風險** | 怕自己有事家庭撐不住、保單買得不清楚、父母醫療、孩子教育金。 |
| **最在意的人** | 太太與兩個孩子。 |
| **相處風格與地雷** | 理性、重邏輯、時間有限、講重點；討厭話術與催促，需被尊重專業判斷。 |

# **產出二：對話準備卡（範例已填寫）**

第五段的答案直接整理成這張卡，業務員帶著它去拜訪客戶即可。

| 準備項目 | 內容（範例：陳志明） |
| :---- | :---- |
| **開場白（不談商品）** | 「謝謝你今天抽空。今天不跟你談任何商品，主要想多了解你跟家庭的狀況，看看有沒有我能幫上忙的地方。」 |
| **可切入的話題** | 工作與團隊近況、孩子、最近同事生病的事、最近在看的房子。 |
| **先問的三個問題** | ① 家裡開銷主要是靠你嗎？　② 之前的保單還記得保了什麼嗎？　③ 如果暫時沒辦法工作，家裡撐得住嗎？ |
| **要避免的地雷** | 別貶低他現有的保單、別一開始就報商品、別催促他做決定。 |
| **想取得的下一步** | 約下次做一份保單健檢、攤開缺口，並邀太太一起參與。 |

| 一句話總結　業務員不需要學 SPIN——他只要好好回答顧問的這幾段問題，客戶輪廓與對話計劃就自然成形了。 |
